# HR Attrition Analytics Power BI Dashboard

## Overview
This Power BI dashboard provides a comprehensive analysis of employee attrition within an organization. By leveraging key HR metrics, the dashboard helps HR professionals and organizational leaders understand patterns and factors contributing to employee turnover.

## Features
- *Attrition Overview:* Visualizations showing overall attrition rates, monthly/quarterly trends, and departmental comparisons.
- *Demographics Analysis:* Insights into attrition rates based on age, gender, tenure, and other demographic factors.
- *Attrition by Department:* Detailed analysis of attrition rates across different departments, job roles, and locations.
- *Exit Reasons:* Analysis of common reasons for employee departures, sourced from exit interviews and surveys.
- *Predictive Analytics:* Use of machine learning models to predict potential attrition risk based on historical data.

## Data Sources
The dashboard integrates data from multiple sources, including:
- *Employee Database:* Contains personal details, job titles, departments, and tenure information.
- *Exit Surveys:* Provides reasons for employee turnover and feedback from departing employees.
- *Performance Management System:* Tracks performance ratings and feedback.
- *HRIS (Human Resource Information System):* Manages payroll, benefits, and employee records.

## How to Use the Dashboard
1. *Download the Dashboard File:*
   - Download the HR_Attrition_Analytics.pbix file from this repository.

2. *Open Power BI:*
   - Open Microsoft Power BI Desktop.

3. *Load the Dashboard:*
   - Open the HR_Attrition_Analytics.pbix file in Power BI Desktop.

4. *Connect to Your Data Sources:*
   - Update the data source settings to point to your specific data sources.
   - Refresh the data to see the latest insights.
## Insights Provided
- *Attrition Rates:* Understand overall and segmented attrition rates, and identify trends over time.
- *Demographic Insights:* Analyze how factors like age, gender, and tenure impact attrition rates.
- *Departmental Analysis:* Compare attrition rates across different departments and identify high-risk areas.
- *Exit Reasons:* Gain insights into the primary reasons for employee departures.
- *Predictive Analytics:* Identify employees at high risk of leaving using predictive models.
